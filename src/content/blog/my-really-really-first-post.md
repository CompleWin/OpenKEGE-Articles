Люди в СССР решали ЕГЭшные задачи по информатике, не спеша, не суетясь, с легкой долей ирониии и чувством, что каждая задача решена не зря, используя функции `all` и `any`. Сейчас про эти чудесные функции узнаете и вы.

Начнем с их синтаксиса. Эти функции принимают какую-то коллекцию, это может быть строка, список, множество. Для упрощения — все, по чему можно итерироваться, то есть по чему можно идти циклом `for`. По целым числам идти в цикле не получится, передать их не удастся. На выходе получаем всего два возможных результата: `True` или `False`.  Python приводит каждый элемент к `bool`. Ложными считаются: `False`, `0`, `0.0`, `None`, `""`, `[]`, `{}`, `()` , всё остальное истинно. `all` возвращает `True`, если все элементы истинны (или коллекция пуста). `any` возвращает `True`, если хотя бы один элемент истинен. То есть эти функции — простой аналог логических операторов `and` и `or` в условиях `if`. Посмотрим на пару примеров:

Проверить, что все числа списка четные

```python
nums = [2, 4, 6, 8, 3]
print(all(x % 2 == 0 for x in nums)) # False
```

Здесь кроется небольшой технический нюанс, который называется [ленивым вычислением](https://pythonist.ru/lenivye-vychisleniya-v-python/). Мы передаем в функцию не список, а генераторное выражение. То есть списка в памяти нет, Python его не создаст. При этом проверяется элемент по одному. Как только `all` встретит ложь, она вернет `False` и оставшиеся элементы проверяться не будут. (У `any` наоборот, если она встретит `True` , то вернет `True` и остальные элементы проверять не будет)

Многие заметят, что это копирка обычного `if` с циклом. Справедливое замечание. Для простых задач использование этих функций кажется бессмысленным. В каких сценариях могут пригодиться эти функции?

К примеру в номере 8, где нас будут просить найти слова, которые состоят из уникального числа символов (все символы различны).

```python
for w in product("ЭПШТЕЙН", repeat=4):
		word = "".join(w)
		if all(word.count(x) == 1 for x in word):
				cnt += 1
```

Теперь пример посложнее, номер 24 и вопрос: найти подстроки, в которых каждая латинская гласная буква встречается ровно 8 раз. То есть только те подстроки, где не все гласные буквы 8 раз, а каждая гласная 8 раз. В теории можно обычным `if` перечислить все эти буквы, их не так много, но можно немного упростить себе жизнь.

```python
vowels = "AEIOUY"
all(s.count(v) == 8 for v in vowels) 
```

Стоит держать в уме то, что на пустой коллекции функция `all([])` вернет `True` , а функция `any([])` вернет `False` .
