# Тонкости JavaScript

В JavaScript есть несколько известных «странностей», которые путают
даже опытных разработчиков. Разберём пару из них.

Чтобы безопасно проверить на null, используй строгое равенство:

<LiveEditor

## Array.map и reduce

Попробуй сам — измени массив или функции:

<LiveEditor
  client:visible
  initialCode={`const numbers = [1, 2, 3, 4, 5];

const doubled = numbers.map(n => n * 2);
const sum = numbers.reduce((a, b) => a + b, 0);

console.log('Удвоенные:', doubled);
console.log('Сумма:', sum);`}
/>
