# OpenKEGE Articles

Проект предоставляет собой написанный на astro сайт с минимальным числом функций. 
Только чтение, и небольшой интерактив в виде квиза и live кодинг сегментов.

## Структура проекта

```text
├── public/
├── src/
│   ├── assets/
│   ├── components/
│   ├── content/
│   ├── layouts/
│   └── pages/
├── astro.config.mjs
├── README.md
├── package.json
└── tsconfig.json
```
